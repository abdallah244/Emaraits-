## الإبلاغ عن ثغرة أمنية

نأخذ أمان مشروع **Emaraits** على محمل الجد. إذا اكتشفت ثغرة أمنية، نرجو اتباع الخطوات التالية:

### ⚠️ مهم: لا تُبلغ عن الثغرات الأمنية عبر Issues العامة

### خطوات الإبلاغ

1. **أرسل بريدًا إلكترونيًا** إلى فريق الأمان على العنوان المُحدد في إعدادات المستودع
2. **قم بتضمين** المعلومات التالية:
   - وصف مفصل للثغرة
   - خطوات إعادة إنتاج المشكلة
   - التأثير المحتمل
   - اقتراحات للإصلاح (إن وجدت)

### ما يمكنك توقعه

| المرحلة         | المدة الزمنية    |
| --------------- | ---------------- |
| تأكيد الاستلام  | خلال 24 ساعة     |
| التقييم الأولي  | خلال 72 ساعة     |
| تحديث عن الحالة | خلال أسبوع       |
| إصدار الإصلاح   | حسب خطورة الثغرة |

### تصنيف الخطورة

| المستوى      | الوصف                           | وقت الاستجابة |
| ------------ | ------------------------------- | ------------- |
| 🔴 **حرج**   | تسريب بيانات، تنفيذ كود عن بُعد | خلال 24 ساعة  |
| 🟠 **عالي**  | تجاوز المصادقة، حقن SQL         | خلال 48 ساعة  |
| 🟡 **متوسط** | XSS، كشف معلومات محدودة         | خلال أسبوع    |
| 🟢 **منخفض** | مشاكل تكوين، تحسينات أمنية      | خلال أسبوعين  |

## الإجراءات الأمنية المُطبقة

### المصادقة والتفويض

- 🔐 مصادقة JWT مع رموز التحديث (Refresh Tokens)
- 🛡️ التحكم في الوصول بناءً على الأدوار (RBAC)
- 🔒 تشفير كلمات المرور بـ bcrypt (12 جولة)
- ⏰ انتهاء صلاحية الجلسات تلقائيًا

### حماية البيانات

- 🔑 تخزين المفاتيح في متغيرات البيئة
- 🔒 اتصالات HTTPS مُشفرة
- 📝 تصفية وتنقيح المدخلات (منع XSS وحقن SQL)
- 🗄️ تشفير البيانات الحساسة في قاعدة البيانات

### حماية الـ API

- 🚫 تحديد معدل الطلبات (Rate Limiting)
- 📋 تكوين CORS للنطاقات المسموحة فقط
- ✅ التحقق من صحة جميع المدخلات
- 📊 تسجيل ومراقبة جميع الطلبات

### أمان التطبيق

- 📌 تثبيت شهادات SSL (Certificate Pinning)
- 🔐 تخزين آمن للبيانات المحلية
- 🛡️ حماية من هجمات MITM
- 📱 تشويش الكود (Code Obfuscation) في الإنتاج

## ممارسات التطوير الآمن

- مراجعة الكود الأمنية قبل كل دمج
- فحص التبعيات بحثًا عن الثغرات المعروفة
- اختبارات أمنية دورية
- تحديث المكتبات والتبعيات بانتظام
